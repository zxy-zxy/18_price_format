# Price Formatter

Python script to format price with a separation by thousands with a space.

# Requirements
Python 3.5 required.
# Usage examples
## Usage with terminal
### Input
```bash
python format_price.py 10450.551
``` 
### Output
```bash
10 450.55
```
## Usage in your own python project 
```python
from format_price import format_price
```
# Tests
